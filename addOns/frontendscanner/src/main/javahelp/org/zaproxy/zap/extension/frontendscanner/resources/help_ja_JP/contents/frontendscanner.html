フロントエンドスキャナー

フロントエンドスキャナー (Front-End Scanner) アドオンがインストールされ有効化されると、ZAP を通過するすべての HTTP レスポンスを操作して、カスタムロジック、ユーティリティ関数、および ZAP ユーザーがブラウザで実行したいスクリプトを含む JavaScript コードを注入します。 注意: アドオンが完全に機能するためには、Webページから CSP ヘッダーを自動的に削除します。この操作はセキュリティリスクを伴う可能性があるため、テスト環境でのみ使用し、本番環境では CSP を無効化しないよう注意してください。

ブラウザで実行されるスクリプトを追加する

注意: 現在、Webページに注入されるのは「クライアントサイド・パッシブ」スクリプトのみです。

パッシブスクリプトを追加するには、「スクリプト」ペインに移動し、「クライアントサイド・パッシブルール」を右クリックして「新規スクリプト...」を選択します。 スクリプトに分かりやすい名前を付け、「scan.js」テンプレートを使用していることを確認し、保存します。

スクリプトはWebブラウザで実行されるため、JavaScript で記述する必要があります。 同じ理由で、使用するブラウザが公開する Web API を利用できます。

ZAP のインターフェース経由で作成された各ユーザースクリプトは、フロントエンドで関数にラップされます。この関数は JavaScript オブジェクト (FrontEndScanner) をパラメーターとして受け取ります。 これにより、スクリプトはこのオブジェクトが公開するものにアクセスできます：

  • ZAP 定数: スクリプトがアラートを作成するのを助けます。
  • 「メールボックス (publish-subscribe メカニズム)」: ブラウザで発生するイベント (ユーザーインタラクション、ストレージアクセスなど) にスクリプトが反応するための仕組み。
  • 発見事項をZAPに報告するためのヘルパー関数
ユーザー (パッシブ) スクリプトを含むすべての関数は、ブラウザで自動的に実行されます。

有効化 - 無効化

アドオンを切り替えるには、「ツール」メニューから「フロントエンドスキャナーを切り替え」を選択します (注意: アドオンはデフォルトでオフになっています)。

注意: 有効化されたスクリプトのみが対象のWebページに注入されます。 特定のスクリプトを有効化 (または無効化) するには、「スクリプト」ペインでそれらを右クリックし、「スクリプトを有効化」 (または「スクリプトを無効化」) を選択します。
