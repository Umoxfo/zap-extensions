Replacer – 自動化フレームワーク対応

このアドオンは自動化フレームワークに対応しています。

ジョブ: requester

replacer ジョブを使用すると、置換ルールを追加できます。

この内容は、動画「ZAP Chat 09 Automation Framework Part 3 - Requests」で解説されています。 

  - type: "replacer"                   # 置換ルール
    parameters:
      deleteAllRules:                  # Boolean、trueの場合、既存のすべての置換ルールを削除します。デフォルト：false
    rules:                             # 置換ルールのリスト (複数のルールを定義可能)
      - description:                   # 文字列、ルールの名前（説明）
        url:                           # 文字列、URLと照合するための正規表現。空の場合はすべてのURLに一致します
        matchType:                     # 文字列、以下のいずれか: req_header, req_header_str, req_body_str, resp_header, resp_header_str, resp_body_str
        matchString:                   # 文字列、置き換える対象を指定します
        matchRegex:                    # Boolean、trueの場合、matchString は正規表現として扱われます。デフォルト: false
        replacementString:             # 文字列、マッチした部分を置き換える新しい文字列
        tokenProcessing:               # Boolean、有効にすると動的トークン ({{RINT}}, {{UUID}}, {{TICKS}}) を使用できます
        initiators:                    # イニシエーターを表す整数のリスト (詳細は replacer ページを参照)
パラメーターの詳細については、replacerページを参照してください。
