OpenAPI – 自動化フレームワーク対応

このアドオンは自動化フレームワークに対応しています。

アドオンはスパイダー実行中に発見したOpenAPI定義を自動的に追加しますが、定義が利用可能な場合は、URLまたはローカルファイルを介して明示的に指定することを推奨します。

targetUrlパラメーターは、「ターゲットURLの形式」と同じ方法で動作します。

ジョブ: openapi

openapi ジョブを使用すると、URLまたはファイル経由でOpenAPI定義をインポートできます。

この内容は、動画「ZAP Chat 11 Automation Framework Part 5 - APIs」で解説されています。 

  - type: openapi                      # OpenAPI定義のインポート
    parameters:
      apiFile:                         # 文字列: OpenAPI定義を含むローカルファイル、デフォルト: null (定義はインポートされません)
      apiUrl:                          # 文字列: OpenAPI定義を含むURL、デフォルト: null (定義はインポートされません)
      context:                         # 文字列: OpenAPI定義をインポートする際に使用するコンテキスト、デフォルト: 最初のコンテキスト
      user:                            # 文字列: 認証に使用するオプションのユーザー (自動化フレームワークの環境設定で定義されている必要があります)
      targetUrl:                       # 文字列: 定義内で指定されたターゲットを上書きするURL、デフォルト: null (上書きなし)
