強制ブラウズタブ

The Forced Browse tab allows you to perform a 強制ブラウズ scan on any of the sites that have been accessed.

Sites can be selected via the toolbar or the サイトタブ.
Any sites that have have been or are currently being scanned are marked in bold in the toolbar Sites pulldown control.
The toolbar provides a set of buttons which allow you to start, stop, pause and resume the scan.
A progress bar shows how far the scan of the selected site has progressed.
The 'Current scans' value shows how many scans are currently active - hovering over this value will show a list of the sites being scanned in a popup.
The toolbar also includes a button to export the displayed content as CSV.

右クリック メニュー

Right clicking on one or more rows will bring up a menu which will allow you to:

以下の処理から除外する:

このメニューには以下のサブメニューがあります：

プロキシ

選択したノードをプロキシから除外します。ZAP経由でプロキシされますが、どのタブにも表示されません。
現在テストしているシステムに関係のないURLを無視するために使用できます。
The nodes can be included again via the セッションプロパティ dialog

スキャナー

This will prevent the selected notes from being actively scanned.
The nodes can be included again via the セッションプロパティ dialog

スパイダー

選択したノードがスパイダーされることを防ぎます。
The nodes can be included again via the セッションプロパティ dialog

コピー

This will copy the selected URLs to the clipboard.

再送信...

This will bring up the 再送信ダイアログ which allows you to resend the request after making any changes to it that you want to.

新しいアラート...

This will bring up the アラートの追加ダイアログ が表示され、このリクエストに対して新しい アラート against this request.

URLをブラウザで開く

選択したノードのURLを既定のブラウザーで開きます。

Anti-CSRFテストフォームを生成

これによりCSRF問題のテスト用に生成されたフォームを提供するURLが開きます。
この機能は、APIが有効化されていて、かつお使いのプラットフォームでJavaがブラウザでURLを開くことをサポートしている場合に限り、POSTリクエストに対して有効になります。
