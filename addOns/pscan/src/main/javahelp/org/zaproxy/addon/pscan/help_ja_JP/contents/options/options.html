オプション

The Passive Scanner add-on provides the following options screens:
     パッシブスキャナー Allows to configure the passive scanner.
     パッシブスキャンルール Allows to configure the passive scan rules.
     パッシブスキャンタグ Allows to configure the tags that are added by the passive scanner.
