Postman サポート

このアドオンを使用すると、ZAPのUIから Postman コレクションをインポートできます。 なお、API やコマンドラインからのインポート機能は現在開発中です。

また、自動化フレームワークに対応しています。

UI

「インポート」メニューに以下の項目が追加されます：
  • Postman コレクションをインポート
コレクション内で定義された変数は、それぞれ定義された値に置き換えられます。 また、ダイアログでは key1=value1,key2=value2,... の形式で、カンマ区切りのキーと値のペアを指定できます。これらの変数は、コレクション内の変数よりも優先的に使用されます。

API

APIに以下の操作が追加されます：
  • ACTION importFile (file)
  • ACTION importUrl (url)

コマンドライン

以下のコマンドラインオプションが追加されます：
  • -postmanfile <filename> : 指定されたファイル名から Postman コレクションをインポートします
  • -postmanurl <url> : 指定されたURLから Postman コレクションをインポートします
  • -postmanendpointurl <url> : Postman コレクション内のベースURLを上書きするためのエンドポイントURL
