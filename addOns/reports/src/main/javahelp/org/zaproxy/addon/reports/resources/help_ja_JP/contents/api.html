レポート生成API

APIに以下の操作が追加されます：

ビュー

  • templates: 利用可能なテンプレートを表示する。
  • templateDetails (template*): 指定されたテンプレートの詳細を表示する。

アクション

  • generate (title*, template*, theme, description, contexts, sites, sections, includedConfidences, includedRisks, reportFileName, reportFileNamePattern, reportDir, display): 指定されたパラメーターでレポートを生成します。
    • title: レポートのタイトル
    • template: レポートテンプレート
    • theme: レポートテーマ
    • description: レポートの説明
    • contexts: レポートに含めるコンテキスト名 ('|' で区切ります)。 例: "Default Context|My Context"
    • sites: レポートに含めるサイトURL ('|' で区切ります)。
    • sections: レポートに含めるセクション ('|' で区切ります)。 セクション名は API ビュー templateDetails (template*) で確認できます。
    • includedConfidences: レポートに含める信頼度 ('|'で区切ります)。 指定可能な値は "False Positive"、"Low"、"Medium"、"High"、および "Confirmed" です。
    • includedRisks: レポートに含めるリスク ('|' で区切ります)。 指定可能な値は "Informational"、"Low"、"Medium"、"High" です。
    • reportFileName: 生成されるレポートのファイル名。 この値は reportFileNamePattern パラメーターを上書きします。
    • reportFileNamePattern: レポートファイル名のパターン。 例: {{yyyy-MM-dd}}-ZAP-Report-[[site]]。
    • reportDir: 生成されたレポートが配置されるディレクトリのパス。
    • display: 生成されたレポートを表示するかどうか。 "true" または "false"。
