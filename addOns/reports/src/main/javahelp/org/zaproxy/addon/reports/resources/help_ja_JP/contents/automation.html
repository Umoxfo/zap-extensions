レポート生成 – 自動化フレームワーク対応

このアドオンは自動化フレームワークに対応しています。

ジョブ: report

report ジョブを使用すると、インストールされている任意のレポートテンプレートを使用してレポートを生成できます。 
  - type: report                       # レポート生成
    parameters:
      template:                        # 文字列: テンプレート ID、デフォルト: risk-confidence-html
      theme:                           # 文字列: テンプレートのテーマ、デフォルト: テンプレートに定義されている最初のテーマ (あれば)
      reportDir:                       # 文字列: レポートが書き込まれるディレクトリ
      reportFile:                      # 文字列: レポートファイル名のパターン、デフォルト: {{yyyy-MM-dd}}-ZAP-Report-[[site]]
      reportTitle:                     # 文字列: レポートのタイトル
      reportDescription:               # 文字列: レポートの説明
      displayReport:                   # Boolean: 生成時にレポートを表示するかどうか、デフォルト: false
    risks:                             # List: このレポートに含めるリスク、デフォルトはすべて
      - high
      - medium
      - low
      - info
    confidences:                       # List: このレポートに含める信頼度、デフォルトはすべて
      - high
      - medium
      - low
      - falsepositive
    sections:                          # List: このレポートに含めるテンプレートセクション - 該当するテンプレートを参照、デフォルトはすべて
    sites:                             # List: このレポートに含めるサイト、デフォルトはすべて
サイトは完全なURLである必要はありません。サイトリスト内の文字列のいずれかを含むサイトはすべて含まれます。 サイトリストは変数もサポートしています。
