オプション

The Network add-on provides the following options screens:
     クライアント証明書 Allows to configure the client certificates, PKCS#11 and PKCS#12.
     接続 Allows to configure the connection.
     グローバル除外 Allows to configure the global exclusions.
     ローカルサーバー/プロキシ Allows to manage and configure the local servers/proxies.
     レート制限 Allows to configure rate limiting for domains and hosts.
     サーバー証明書 Allows to manage and configure the root CA certificate and issued certificates.
