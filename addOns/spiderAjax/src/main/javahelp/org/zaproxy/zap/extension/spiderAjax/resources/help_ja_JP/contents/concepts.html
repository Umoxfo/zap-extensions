AJAXスパイダー

AJAXスパイダーアドオンは、Crawljax と呼ばれる AJAX リッチサイトのクローラーをZAPに統合します。 これを使用して、ターゲットサイトのページを特定できます。 より良い結果を得るために、(通常の) スパイダーと組み合わせることができます。

The spider is configured using the Options AJAX Spider screen.

このアドオンは自動化フレームワークに対応しています.
