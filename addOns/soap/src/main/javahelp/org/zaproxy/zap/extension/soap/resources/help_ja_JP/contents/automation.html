SOAP – 自動化フレームワーク対応

このアドオンは自動化フレームワークに対応しています。

アドオンは、スパイダリング中に見つかった場合、SOAP エンドポイントを含むWSDLファイルをインポートしますが、利用可能であればURLまたはローカルファイル経由で明示的に追加することを推奨します。

ジョブ: soap

soap ジョブは、ローカルまたはURLからWSDLファイルをインポートできます。

この内容は、動画「ZAP Chat 11 Automation Framework Part 5 - APIs」で解説されています。 

  - type: soap                         # SOAP WSDL インポート
    parameters:
      wsdlFile:                        # 文字列: WSDL のローカルファイルパス、デフォルト: null、定義はインポートされません
      wsdlUrl:                         # 文字列: WSDL を指すURL、デフォルト: null、定義はインポートされません
